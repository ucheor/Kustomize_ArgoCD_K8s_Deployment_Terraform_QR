Automated Cluster Provisioning & GitOps Integration

Project Overview

This project demonstrates a production-ready approach to cloud infrastructure. By utilizing Infrastructure as Code (IaC), we provision fully-functional Kubernetes clusters that come pre-configured with GitOps controllers from the moment they are created.

Infrastructure as Code

Modular Terraform scripts manage EKS/K8s resources, ensuring automated provisioning and trackable infrastructure management.

Bootstrapped GitOps

Clusters are integrated with ArgoCD upon creation. Using the Helm provider within Terraform, the GitOps framework is established from the get go.

Declarative Deployments

Application lifecycles are managed via Git repositories. Any change to the manifest results in an automatic synchronization to the cluster.
